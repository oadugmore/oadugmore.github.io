Sync,hue

A color-coordinated platforming game.

View

Sync,hue is my latest game development project. It is a platforming game based around the idea of synchronized, color-coordinated movement. The player uses two buttons — a blue button and an orange button — to control objects of the corresponding color. Other objects have various other colors which provide different behaviors. This allows for a unique combination of platforming and puzzle elements that create a fun, challenging experience.

I am using the Unity 3D game engine, writing game logic in C#, and creating my own 3D models in Blender and Unity's own ProBuilder.

A beta version of the game is available on Android and iOS.

In The Know

Stay in the know about the situations around you.

View

In The Know is a mobile app that I originally created for CS 4990 at Cal Poly Pomona, and I am still actively developing it. The app shows you information about potentially dangerous or disruptive situations, using the power of Twitter and natural language processing. The app can also be configured to show periodic notifications when a situation is detected matching the user's search phrase.

I am using the Flutter framework for the client app, and a custom Python function hosted in Google Cloud for the app's main functionality. The Google Cloud function fetches data through the Twitter API, and processes the data using the Python NLP library spaCy. I use a modified version of the standard English model with custom named entity recognition rules for detecting various types of situations.

The app is currently available on Android, with a beta version also available for iOS.
